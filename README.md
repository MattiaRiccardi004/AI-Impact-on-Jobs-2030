# AI Impact on Jobs 2030

Questo repository contiene il materiale utilizzato per l'analisi del dataset **"AI Impact on Jobs 2030"**.

## Contenuto del repository

* `Presentazione_AI_Jobs_UNIMI.pptx`: presentazione PowerPoint che illustra i risultati dell'analisi e i principali insight emersi dal dataset.
* `AI_jobs_UNIMI.py`: script Python utilizzato per l'elaborazione dei dati e la generazione dei grafici inclusi nella presentazione.

## Dataset

Il dataset utilizzato è disponibile su Kaggle al seguente link:

https://www.kaggle.com/datasets/khushikyad001/ai-impact-on-jobs-2030/data

## Obiettivo del progetto

L'obiettivo dell'analisi è esplorare il potenziale impatto dell'intelligenza artificiale sul mercato del lavoro entro il 2030, evidenziando:

* le professioni più esposte all'automazione;
* le cause che posso indurre un lavoro ad essere considerato rimpiazzabile dall'AI.
